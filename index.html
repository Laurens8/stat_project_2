Hallo ik ben Laurens De Wit

En welkom op mijn webpagina

Als student programmeren ben ik gemotiveerd en enthousiast om in de IT wereld te stappen. Hoewel ik nog geen ervaring heb in de IT sector bied ik creativiteit en leer ik snel bij. Ik hoop om te kunnen groeien in zowel vaardigheid en als een persoon in mijn opleiding programmeren.

In de richting programmeren leren we applicaties maken met het computertaal C#. In de richting webdesign leren we werken met de computertalen HTML, CSS en javascript. En we leren ook datamodelleren met de computertaal SQL en leren we werken met NoSQL database program MongoDB.

Momenteel zit ik nog maar in het eerste jaar in de richting programmeren en moet ik nog veel bij leren, check zeker geregeld mijn github repositories zodat je mijn progressie in het programmeren kunt mee volgen.
